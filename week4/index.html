Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

Traditionally, when a form was submitted, it would be sent to a server where the information would be processed using a the backend. It is also possible to process the form in the client before sending it to the server, for example, if we wanted to provide one extra layer of validation to our form.

A form object also has a method called elements that returns an HTML collection of all the elements contained in the form
