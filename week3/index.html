Objects

JavaScript is designed on a simple object-based paradigm. An object is a collection of properties, and a property is an association between a name (or key) and a value. A property's value can be a function, in which case the property is known as a method. â§‰

This

Keyword used to refer to the current object in a method or constructor. A common use is when you want to access an attribute of the object.

It can be used to:

  • Invoke current class constructor
  • Invoke current class method
  • Return the current class object
  • Pass an argument in the method call
  • Pass an argument in the constructor call

Array Cardio

.filter(): used to filter elements of a given array.

- Callback arguments: item, index, list.

- Return: a new array filled with elements that pass a test provided by a function.


.map(): calls a function for every item of the given array.

- Callback arguments: item, index, list.

- Return: a new array from calling a function for every array element.


.reduce(): uses a function to reduce the whole array to a single value.

- Callback arguments: result, item, index, list.

- Return: a single value: the function's accumulated result.


.forEach(): for each item of the array: do something with it.

- Callback arguments: item, index, list.


.sort(): arranges the elements using the specified criteria.

- Callback function: the function should return a negative, zero, or positive value, depending on the arguments.

- Callback arguments: previousItem, currentItem.

- Be careful, the .sort() method overwrites the original array.


.find(): used to find one specific element.

- Callback arguments: item, index, list.

- Return: the value of the first element that passes a test.


.findIndex(): finds the index of one specific element.

- Callback arguments: item, index, list.

- Return: the index (position) of the first element that passes a test.
